# Assets Sport

Images de guides de mouvement fournies par l'utilisateur dans Codex pour la tranche Sport isolee.

- `push-wall-guide.png` : pompes au mur.
- `push-knees-guide.png` : pompes sur les genoux.
- `push-standard-guide.png` : pompes classiques.

Ces assets remplacent les illustrations schematiques internes uniquement pour les variantes correspondantes. Les autres mouvements gardent le fallback SVG local. Avant publication hors MVP, confirmer le statut de licence et de validation finale des images.
